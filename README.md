## 🎯 About

**ViewBeforeBuy** (formerly ImmoFin AI) solves a critical problem in the Moroccan real estate market:

> Buying a property in Morocco takes **6 to 18 months** on average — due to financial opacity, lack of digital tools, and no 3D visualization for buyers.

Our solution combines **three pillars** into one unified platform:

| Pillar | Description |
|--------|-------------|
| 🤖 **AI** | Conversational assistant (voice & text) powered by Claude API |
| 💻 **Digital** | Interactive 3D visualization of properties |
| 💰 **Fintech** | Credit simulator, bank comparator, eligibility scoring |

---

## ✨ Features

### 🏢 Agency Dashboard (B2B)
- **AI-powered property creation** — Agent speaks to the AI, property is generated automatically with 3D visualization
- **3D Property Viewer** — Interactive rotation, zoom, interior/exterior views (Villa, Apartment, Riad, Duplex)
- **Document Upload** — PDF, JPG, DWG plan upload → AI generates complete property description
- **AI Voice & Text Assistant** — Real-time responses to client questions
- **Conversation History** — Full chat history with each client interaction
- **Analytics Dashboard** — Virtual visits, interested clients, simulated credits

### 📱 Client App (B2C)
- **AI Financial Report** — Receive complete financial report generated by agency AI
- **Credit Simulator** — Real-time mortgage calculation (rate, duration, monthly payment)
- **Bank Comparator** — Compare CIH Bank, Attijariwafa, BMCE Bank offers side by side
- **Eligibility Score** — 🟢 Eligible / 🟡 Borderline / 🔴 Insufficient
- **Credit Tracking** — Follow active credits, payment history, upcoming deadlines
- **AI Advisor** — Personalized financial advice and recommendations

## 🚀 Getting Started

### Prerequisites
```bash
Node.js >= 18.0.0
npm >= 9.0.0
```

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/ViewBeforeBuy.git

# Navigate to project directory
cd ViewBeforeBuy

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
```

### Environment Variables

Create a `.env` file in the root directory:

```env
VITE_ANTHROPIC_API_KEY=your_claude_api_key
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm run preview
```
